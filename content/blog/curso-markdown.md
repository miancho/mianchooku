## Headings

El # reemplaza a la H, y la cantidad es el tamaño

### Ejemplos

# Title -> H1
## Title -> H2
### Title -> H3
#### Title -> H4
##### Title -> H5
###### Title -> H6

## Listas
Con sólo el asterisco creamos la lista con viñetas.

* Manzanas
* Bananas
* Peras
* Frutillas
* Durazno

Con sólo el número y punto creamos la lista ordenadas.

1. Italia
2. Francia
3. Argentina
4. España
5. Estados Unidos

Sub-items para listas desordenadas de la siguiente manera.

* Manzanas
  * Verde
  * Rojas
* Bananas
* Peras

y para las listas ordenadas

1. Italia
   1. Pueblos
   2. Ciudades
2. Francia
3. Argentina
4. España
5. Estados Unidos
